OBC

On-Board Computer

La On-Board Computer (OBC) de un satélite es el sistema que gestiona y controla sus operaciones. Procesa datos de sensores, ejecuta comandos y coordina la comunicación entre subsistemas, asegurando un funcionamiento autónomo y tomando decisiones críticas para la misión. Además, almacena y transmite datos a la estación terrestre.

Sobre FiubaSAT

FiubaSAT es un proyecto de CubeSAT que tiene como objetivo fomentar el desarrollo y aprendizaje tecnológico orientado a actividades espaciales. Se plantea desde el comienzo un enfoque de trabajo de código Open Source con metodología DevOps para que sirva a futuros contribuyentes a sumarse fácilmente y plantear trabajos profesionales en torno a este proyecto.
